# Binary Search Tree

A simple generic Binary Search Tree implementation in Go.

A Binary Search Tree, also known as BST, is a tree data structure where each node follows this rule:

- Values smaller than the node go to the left subtree
- Values greater than the node go to the right subtree

---

## Features

- Generic Binary Search Tree
- Supports ordered data types
- Insert values
- Search values
- Delete values
- Find minimum value
- Find maximum value
- In-order traversal
- Pre-order traversal
- Post-order traversal
- Check tree height
- Check tree size
- Clear the tree
